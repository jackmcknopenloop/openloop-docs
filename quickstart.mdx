# Your First Integration in 10 Minutes

Ready to transform your practice? This guide will have you connected and referring your first patient to specialized care in under 10 minutes.

## What You'll Need

- **API Key** - Get one from our partnership team
- **HTTP Client** - curl, Postman, or your favorite tool
- **Basic API Knowledge** - You've got this! ðŸ’ª

## Step 1: Verify Your Connection

Let's start by checking that your API credentials are working:

```bash
# Production
curl -H "x-api-key: YOUR_API_KEY" \
     https://partners-api-sigma.vercel.app/health

# Staging (for testing)
curl -H "x-api-key: YOUR_API_KEY" \
     https://staging-partners-api-sigma.vercel.app/health
```

## Step 2: Explore Available Programs

Before referring patients, let's see what programs are available:

```bash
curl -H "x-api-key: YOUR_API_KEY" \
     https://partners-api-sigma.vercel.app/programs
```

This will return a list of available medical programs your patients can access.

## Step 3: Create Your First Patient

Now let's create a patient record:

```bash
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "patient@example.com",
    "first_name": "John",
    "last_name": "Doe",
    "phone": "+1234567890",
    "birth_date": "1990-01-01",
    "gender": "male",
    "sex": "male",
    "locations": [{
      "address_line_1": "123 Main St",
      "city": "Anytown",
      "state": "CA",
      "country": "US",
      "zip_code": "12345",
      "default": true,
      "type": "shipping"
    }]
  }' \
  https://partners-api-sigma.vercel.app/patients
```

## Step 4: Enroll Patient in a Program

Once you have a patient, you can enroll them in a medical program:

```bash
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "program_id": 1,
    "stripe_customer_id": "cus_example123",
    "stripe_product_id": "prod_example456"
  }' \
  https://partners-api-sigma.vercel.app/patients/PATIENT_ID/enrollments
```
