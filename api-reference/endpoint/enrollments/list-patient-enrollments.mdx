---
title: 'List Patient Protocol Enrollments'
openapi: 'GET /patients/{patientId}/enrollments'
---

Retrieve all protocol enrollments for a specific patient from the OpenLoop Health Partners API. Each protocol enrollment automatically manages the underlying program enrollments.

**Key Features:**
- View all protocol enrollments for a patient
- See associated program enrollments created by each protocol
- Access enrollment status and metadata

**How It Works:**
- **Protocol Enrollments**: High-level enrollment in a protocol
- **Program Enrollments**: Individual program enrollments created automatically by protocols
