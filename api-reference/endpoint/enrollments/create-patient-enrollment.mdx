---
title: 'Create Patient Enrollment'
openapi: 'POST /patients/{patientId}/enrollments'
---

Enroll a patient in a healthcare program via the OpenLoop Health Partners API. Creates a new enrollment connecting patient to program for service access.

**Required Fields:**
- `program_id` - Program ID (get from `/programs`)
- `stripe_customer_id` - Stripe customer ID for billing
- `stripe_product_id` - Stripe product ID for the program
