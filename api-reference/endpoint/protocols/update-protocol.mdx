---
title: 'Update Protocol'
openapi: 'PUT /protocols/{protocolId}'
---

Update an existing protocol's configuration in the OpenLoop Health Partners API. Modify program associations, intake form metadata, and other protocol settings.

**Updateable Fields:**
- Protocol name and description
- Program associations (`program_ids`)
- Intake form metadata
- Standard vs custom protocol type

See [Data Standards](/api-reference/data-standards) for formatting rules and validation patterns.
