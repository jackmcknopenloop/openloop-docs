---
title: 'Reset Database'
openapi: 'POST /admin/reset'
---

Reset the database to its initial state, removing all data including patients, protocols, and enrollments. This endpoint is useful for development and testing environments.

**Warning:** This action permanently deletes all data in the database. Use with extreme caution in production environments.

**Use Cases:**
- Development environment cleanup
- Testing data isolation
- Fresh start for demonstrations
